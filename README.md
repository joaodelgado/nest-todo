## Description

Demo TODO app backend built with [NestJS](https://nestjs.com/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# tests for each main module
$ npm run test:domain
$ npm run test:api
$ npm run test:db

# test coverage
$ npm run test:cov
